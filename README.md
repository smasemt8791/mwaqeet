 **تطبيق مواقيت الصلاة – Mawaqeet PWA**  
تطبيق ويب تقدمي عالي الأداء لعرض مواقيت الصلاة العالمية بدقة متناهية

#### نبذة عن المشروع
«مواقيت» هو تطبيق ويب تقدمي (Progressive Web App) تم تطويره لتوفير مواقيت الصلاة اليومية بدقة عالية جدًا في أي مكان بالعالم. يُعد هذا المشروع نموذجًا عمليًا متقدمًا لتطبيقات PWA، تم تطويره بواسطتي وإن لم يكن له علاقة مباشرة بالصحة وتطبيقاتها الا أنه يعكس الالتزام التام ببناء حلول رقمية قوية، سريعة الاستجابة، وقابلة للعمل بشكل كامل دون اتصال بالإنترنت (Offline-First).

#### الميزات الرئيسية
- دقة فائقة في حساب المواقيت بناءً على الموقع الجغرافي وطرق الحساب المعتمدة عالميًا  
- تجربة Offline-First كاملة: يعمل التطبيق بسلاسة تامة دون أي اتصال بالإنترنت بعد التثبيت الأول  
- تصميم متجاوب 100% يتناسب مع جميع الأجهزة (هواتف – تابلت – حواسيب)  
- دعم كامل للغة العربية بنظام الكتابة من اليمين لليسار (RTL) مع إمكانية التبديل الفوري للإنجليزية  
- إشعارات يومية اختيارية لكل أذان (بإذن المستخدم)  
- قابل للتثبيت على الشاشة الرئيسية كتطبيق أصلي تمامًا دون الحاجة لمتاجر التطبيقات  

#### التقنيات المستخدمة
| التقنية             | الدور والفائدة |
|---------|----------------|
| TypeScript (أكثر من 95%) | كود آمن، مضبوط الأنواع، سهل الصيانة والتوسع |
| React   | واجهات مستخدم تفاعلية عالية الكفاءة والأداء |
| Vite    | أسرع أداة تطوير وبناء في الوقت الحالي |
| Tailwind CSS | تصميم سريع ومتجاوب وواجهة أنيقة وخفيفة |
| 
| PWA + Workbox | تجربة تطبيق أصلي، عمل بدون إنترنت، إشعارات، تثبيت |
| مكتبات حساب المواقيت الموثوقة | ضمان أعلى درجات الدقة والاعتمادية |

#### تعليمات التشغيل للمطورين

```bash
cd mawaqeet-pwa
npm install
npm run dev    # تشغيل وضع التطوير
npm run build   # بناء النسخة النهائية
```

 **Mawaqeet – Prayer Times PWA**  
High-Performance Progressive Web App for Accurate Worldwide Prayer Times

#### Project Overview
Mawaqeet is a modern Progressive Web App (PWA) that provides highly accurate prayer times anywhere in the world. While developed by me and not directly related to health and its applications, it reflects a complete commitment to building robust, highly responsive, and fully functional digital solutions offline.

#### Key Features
- Extremely accurate prayer time calculation based on geographic coordinates and globally recognized methods  
- Complete Offline-First experience – works perfectly without internet after first install  
- Fully responsive design optimized for phones, tablets, and desktops  
- Full Arabic (RTL) support with instant language switching to English (LTR)  
- Optional daily notification reminders for each prayer (with user permission)  
- Installable on home screen like a native app – no app store not required  

#### Tech Stack
| Technology            | Purpose & Benefit |
|-----------------------|-------------------|
| TypeScript (95%+)     | Type-safe, maintainable, and scalable codebase |
| React                 | High-performance interactive UI components |
| Vite                  | Lightning-fast development and build tool |
| Tailwind CSS          | Rapid, responsive, and elegant styling |
| PWA + Workbox         | Native-like experience, offline functionality, push notifications |
| Trusted Prayer Libraries | Maximum accuracy and reliability |
